# TestToys
测试人员的工具箱，提升效率。Tester toolbox to maximize productivity.

## 测试用例管理

功能
- 支持导入：从其他测试用例管理软件导出的数据，可导入到本工具中
- 快捷搜索：同时搜索用例编号、用例标题、前提条件、操作步骤、预期结果、备注等字段
